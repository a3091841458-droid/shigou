# Shigou (四郷)

# Shigou is a text-based, atmospheric exploration RPG built in Twine (Harlowe 3). Set in a decaying, snow-buried former coal mining city, the game blends resource management, exploration, and rich historical narrative. As one of the few remaining youths in the city, you must manage your time and stamina to uncover the lost memories (Scraps) of Shigou's golden age before the harsh winter completely erases them.

# 

# ❄️ Overview

# Genre: Interactive Fiction / Text-Adventure RPG

# 

# Engine: Twine (Harlowe 3.3.x)

# 

# Theme: Post-Industrial Decay, Melancholy, Exploration, Memory

# 

# In Shigou, you balance a mundane daily routine as a government worker with the freedom of weekend exploration. The city is a sprawling labyrinth of abandoned arcades, crushed houses, and silent factories. By exploring these ruins, you will find "Scraps"—residual memories left behind by former residents, miners, and the indigenous Ainu people.

# 

# ⚙️ Core Mechanics

# This game features several robust, underlying gameplay systems that go beyond a standard "Choose Your Own Adventure" format:

# 

# Dynamic Time System: Every action (traveling, searching, talking) consumes time. The game tracks the current day of the week, hours, and minutes. If the clock hits 8:00 PM (20:00), the freezing temperature forces an immediate end to your day.

# 

# Stamina \& Survival: Exploring the freezing ruins drains your Stamina. If your Stamina drops to 0, you collapse and are forced to sleep. You can recover Stamina by resting, eating warm food (like Ramen), or visiting the Hot Springs.

# 

# Spatial Navigation \& Travel Time: Moving between locations calculates distance dynamically based on a hidden coordinate system (X, Y), applying time costs based on the distance and your current movement speed.

# 

# Collection \& Progression: There are 30 unique "Scraps" to collect. Gathering them not only unlocks deep lore and psychological imprints from the past but also grants passive buffs (e.g., +Max Stamina, +Movement Speed).

# 

# The Figure Hint System: An interactive, in-game hint system on the player's desk that subtly guides you to undiscovered Scraps.

# 

# 📖 Setting \& Lore

# Shigou is loosely inspired by real-world declining mining cities in Hokkaido. It was built upon four ancestral Ainu villages (hence the name "Four Villages"). The game explores the clash and eventual silence of heavy industrial civilization, the tragedy of a shrinking population, and the quiet beauty of a city returning to nature.
